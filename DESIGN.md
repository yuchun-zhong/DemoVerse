# DESIGN.md - DemoVerse 设计规范

## 气质与意象
- **关键词**：深夜剪辑室、监视器阵列、帧流转、专业工作台
- **具象场景**：深色背景如多屏监视器墙，中央操作区发出荧光，进度条如时间线流动，生成的视频帧在暗色画布上逐帧显现

## 配色方案
- **背景主色**：#0A0E17（深空蓝黑，模拟监视器关屏底色）
- **卡片/面板**：#131820（略浅的深灰蓝，层次区分）
- **边框/分割**：#1E2533（低对比度描边）
- **主强调色**：#00FFB2（荧光青绿，代表「生成中/活跃」，时间线高亮）
- **次强调色**：#6366F1（靛蓝紫，代表「AI/智能」能力标识）
- **成功态**：#22C55E（标准绿，视频生成完成）
- **错误态**：#EF4444（标准红）
- **正文色**：#E2E8F0（浅灰白）
- **次要文字**：#94A3B8（中灰）

## 字体排版
- **标题字体**：Inter (Google Fonts)，700/900 weight
- **正文字体**：Inter，400/500 weight
- **代码/URL**：JetBrains Mono (Google Fonts)，400 weight
- **排版节奏**：标题大号加粗，与正文形成强对比；URL 输入框使用等宽字体

## 动效与交互
- **运动性格**：流畅、精准，像视频编辑器的时间线滚动
- **生成进度**：步骤式进度条，带荧光色流光动画
- **按钮悬停**：微弱辉光扩散（box-shadow 发散）
- **页面加载**：标题文字逐字显现动画
- **缓动曲线**：cubic-bezier(0.4, 0, 0.2, 1)（Material 标准缓出）

## 视觉策略
- **图形语言**：圆角矩形卡片 + 微弱玻璃态（backdrop-filter: blur）
- **背景**：多层径向渐变 + 微弱网格线纹理，模拟剪辑软件时间线网格
- **图标风格**：线性图标（Lucide Icons），2px 描边，荧光色填充

## 设计禁忌
- 禁止纯白背景或大面积浅色
- 禁止科技蓝（#3B82F6）作为主色
- 禁止彩虹渐变或过度装饰
- 禁止圆角过大（超过 16px），保持专业感
- 禁止动画卡顿或超过 300ms 的延迟感
