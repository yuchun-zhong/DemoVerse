# DESIGN.md - DemoVerse 设计规范

## 气质与意象
- **关键词**：极简黑、开发者工具感、克制高端、Vercel + Linear 设计语言
- **具象场景**：纯黑背景上只有微弱的紫色光晕，中央巨型输入框如同开发者终端，一切元素极度克制、只留必要信息

## 配色方案
- **背景**：#000000（纯黑，Vercel 标志性黑底）
- **卡片/面板**：#0A0A0A（极微弱层次区分）
- **背景提升**：#111111（下拉菜单、hover 态）
- **边框**：#1A1A1A（几乎不可见的描边）
- **边框聚焦**：#333333
- **主强调色**：#8B5CF6（紫色，Vercel 风格渐变 + 进度条）
- **强调浅色**：#A78BFA
- **强调深色**：#7C3AED
- **成功态**：#22C55E
- **错误态**：#EF4444
- **正文色**：#EDEDED
- **次要文字**：#888888
- **三级文字**：#555555
- **按钮**：#FFFFFF（纯白，与黑底形成极致对比）
- **顶部光晕**：rgba(139, 92, 246, 0.12)

## 字体排版
- **字体族**：Inter (Google Fonts CN)
- **代码/URL**：JetBrains Mono
- **标题**：48px/700，letter-spacing: -0.03em
- **正文**：15px/400
- **导航**：13px
- **标签**：12px uppercase

## 动效与交互
- **运动性格**：极度克制、精准快速
- **进度条**：紫色渐变 + 微弱流光 shimmer 动画
- **按钮悬停**：微弱 opacity 变化，0.15s
- **下拉菜单**：0.15s 淡入 + 4px 上移
- **缓动曲线**：cubic-bezier(0.4, 0, 0.2, 1)
- **最大动画时长**：200ms

## 视觉策略
- **图形语言**：极简圆角矩形（8px），无装饰
- **背景**：纯黑 + 顶部径向紫色光晕，无网格无纹理
- **导航栏**：毛玻璃 backdrop-filter: blur(12px)
- **输入框**：巨型居中，右侧内嵌白色按钮
- **图标**：线性 SVG，14-20px，低饱和

## 设计禁忌
- 禁止非黑色背景
- 禁止大面积彩色或渐变背景
- 禁止圆角超过 12px
- 禁止动画超过 200ms
- 禁止阴影或发光效果（进度点除外）
- 禁止花哨装饰或多余视觉元素
- 禁止使用科技蓝（#3B82F6）
